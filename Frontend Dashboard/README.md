# Animal Protect Admin Dashboard

## Technologies

- Language: Javascript
- Framework: ReactJs
- Material UI: Ant design

## Getting started

- Install yarn

```
npm install -g yarn
```

- Install dependencies

```
yarn install
```

- Run the project in development

```
yarn start
```
